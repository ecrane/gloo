VERSION VERB
  NAME: version
  SHORTCUT: v

  DESCRIPTION
    Show the application version information.

    This is the same as showing the version by running gloo
    with the --version command line parameter.

  SYNTAX
    version

  PARAMETERS
    None

  RESULT
    None

  ERRORS
    None

  EXAMPLE

    > version
    > v

    This is the same as running this from the command line:

    > gloo --version
